태그(tag)

태그(tag)로 감싸여진 부분이 어떤 역할을 하는지 알려주는 기능이다. 사용 방법은 <p>내용 작성<p> 이런 식으로 작성합니다. '<>' 사이에는 태그의 기능이나 역할을 알려주는 태그명이 들어갑니다. 태그로 감싸여진 부분이 해당 태그의 기능이 발휘되는 곳이다. 앞에있는 태그를 '열리는 태그', 뒤에 있는 태그를 '닫히는 태그' 라고 부릅니다. 닫히는 태그는 '/'로 구분합니다.


자주 쓰는 태그(tag)들

<strong> </strong>: 글씨를 굵게 만듭니다.
<u> </u>: 감싸여진 부분에 밑줄을 넣습니다.
<h1~6> </h1~6>: 본문 제목을 나타내는 태그, 1~6단계가 존재합니다.
<br>: 줄바꿈(New line)을 합니다. 닫히는 태그는 없습니다.
<p> </p>: 본문의 단락 설정을 해줍니다. 번거롭게 <br>태그를 많이 쓰는것 보다 더 간결하게 작성할 수 있어 용이합니다.
<img src="img/image.jpg(이미지 주소)" width="30%">: 이미지 넣기, 소스(stc)는 같은 프로젝트 폴더 안에 있는 소스(image.jpg)를 이용합니다. width는 비율입니다.


목차

<li> </li>: 목차를 만드는 태그 입니다.
<ul> </ul>: 목차를 구분지을 떄 사용하는 태그입니다. 이 안에 감싸여진 부분에 있는 목차들이 서로 연관 목차입니다. 다른 목차간에 <br>태그로 구분짓는 기능과 동일합니다. <ul>태그는 <li>의 부모 태그입니다.
<ol> </ol>: 목차에 넘버링을 부여합니다. <ul>태그와 동일하게 목차간 구분짓는 기능을 합니다. 이 태그 역시 <li>의 부모 태그입니다.


중요한 태그들

<!doctype html>: html 문서라는 것을 나타냅니다.
<html> </html>: 헤드와 바디를 감싸는 최고 위 태그입니다. html 문서라는 것을 나타냅니다.
<head> </head>: 본문을 설명하는 부분을 나타내는데 사용하는 태그입니다.
<body> </body>: 본문 부분을 나타내는데 사용하는 태그입니다.
<a> </a>: 감싸여진 부분에 링크를 거는 태그입니다.
<a href="http://www.w3.opg/TR/html51/" target="_blank" title="html5">: 태그로 감싸여진 부분에 링크를 걸어둡니다. target="_blank"는 새탭 열기를 합니다.
