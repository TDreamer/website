기본 문법


#include

#include 라는 것은 특정 함수 등을 사용하기 위해서 헤더파일을 포함하는 것입니다.

위의 코드는 iostream 이라는 파일을 포함하겠다. 라는 뜻이며 input output stream 을 줄여서 iostream 이라는 명칭을 붙였습니다. 뜻 그대로 입력, 출력 명령어를 사용할 수 있게끔 해줍니다. C++의 주요 헤더 파일입니다. (C언어의 stdio.h 이상)


using namespace std

std 라는 네임스페이스를 사용하겠다. 라는 뜻이며 뜻 그대로 이름공간을 사용한다는 것입니다. 아래의 cout과 endl은 std 라는 이름공간 안에 있는 것이만 이름공간을 사용한다고 했으니 사용할 수 있는 것입니다. 만약 using namespace std; 구문이 없다면 아래처럼 오류가 납니다.


cout endl

cout은 console out 을 줄여서 cout으로 한 것이고 말 그대로 콘솔에 출력 이라는 뜻으로 우리가 보는 콘솔창에 출력하라는 뜻입니다. 출력하고 싶은 내용이 있으면 cout의 오른쪽에 내용을 적어주면 됩니다. 위 구문의 경우 오른쪽에 "Hello, World" 라고 적혀있습니다. 이는 Hello, World 라는 글자들을 "" 큰따옴표로 감싸서 문자열 이라는 것으로 만들어 출력하라는 것입니다. 기본적으로 컴퓨터가 인식할 수 있는 자료형 외에는 출력이 되지 않습니다

주석

c++ 코드는 결국엔 사람이 작성하는 것입니다. 메모장에 작성해도 되고 비주얼스튜디오 같은 IDE 에서 작성해도 되는데 작성하다보면 내가 나중에 확인해야될 정보가 있을 수도 있고 다른사람들에게 알려주고 싶을때도 있는데 이런것들을 다른 워드나 파일에 정리하는 것도 좋지만 코드 바로 옆에 주석의 형태로 작성 할 수 있습니다. 주석은 코드에 존재하나 프로그램이 만들어지는데 영향을 끼치지 않습니다. c++에는 주석 작성법이 2가지가 있습니다.

한 줄만 주석처리를 하고 싶으면 위 코드의 7행에 //내용 과 같이 작성하고, 여러줄 주석처리를 하고싶으면 9-12행에 보이듯이 /* 내용 */ 과 같이 작성하면 됩니다.
